BucketzAPI

A REST API for getting user bucket information

Verb Route Description
GET /api/users See all the users
POST /api/users Post a new user
GET /api/users/:userId Get one user
PUT /api/users/:userId Update one user
DELETE /api/users/:userId Delete one user
GET /api/users/fbid/:fbId Get one user by FB id(senderId)
PUT /api/users/fbid/:fbId Update one user by FB id
DELETE /api/users/fbid/:fbId Delete one user by FB id
